= computer
:page-toclevels: 1

`Neutralino.computer` 命名空间包含与用户硬件相关的方法。

[[computergetmemoryinfo]]
== computer.getMemoryInfo()

返回系统内存统计信息，单位为字节。

[[return-object-awaited]]
=== 返回对象（异步）：

* `physical` 对象：物理内存信息。
** `total` 数字：总物理内存。
** `available` 数字：可用物理内存。
* `virtual` 对象：虚拟内存信息。
** `total` 数字：总虚拟内存。
** `available` 数字：可用虚拟内存。

[source,javascript]
----
let memoryInfo = await Neutralino.computer.getMemoryInfo();
console.log(`RAM size: ${memoryInfo.physical.total}B`);
----

[[computergetarch]]
== computer.getArch()

返回CPU架构标识符：`x64` (x86 64位/arm64)，`ia32` (x86 32位)，`arm`，`itanium`，或``unknown``。

[[return-string-awaited]]
=== 返回字符串（异步）：

CPU架构。

[source,javascript]
----
let arch = await Neutralino.computer.getArch();
console.log(arch);
----

[[computergetkernelinfo]]
== computer.getKernelInfo()

返回操作系统内核信息。

[[return-object-awaited-1]]
=== 返回对象（异步）：

* `variant` 字符串：内核类型：`Linux`，`Darwin`，`Windows NT`，或``Unknown``。
* `version` 字符串：版本格式为``<major>.<minor>.<patch>-<build_number>``。

[source,javascript]
----
let kernelInfo = await Neutralino.computer.getKernelInfo();
console.log(`Kernel: ${kernelInfo.variant}`);
----

[[computergetosinfo]]
== computer.getOSInfo()

返回操作系统信息。

[[return-object-awaited-2]]
=== 返回对象（异步）：

* `name` 字符串：操作系统名称。
* `description` 字符串：操作系统描述。
* `version` 字符串：版本格式为``<major>.<minor>.<patch>-<build_number>``。

[source,javascript]
----
let osInfo = await Neutralino.computer.getOSInfo();
console.log(`OS: ${osInfo.name}`);
----

[[computergetcpuinfo]]
== computer.getCPUInfo()

返回CPU信息。

[[return-object-awaited-3]]
=== 返回对象（异步）：

* `vendor` 字符串：厂商名称。
* `model` 字符串：型号名称。
* `frequency` 数字：当前CPU频率，单位为赫兹（Hz）。
* `architecture` 字符串：CPU架构名称。返回与getArch函数相同的值。
* `logicalThreads` 数字：并行模型中的逻辑线程数。
* `physicalCores` 数字：CPU中的物理核心数。
* `physicalUnits` 数字：主板上的物理CPU硬件单元数。

[source,javascript]
----
let cpuInfo = await Neutralino.computer.getCPUInfo();
console.log(`CPU model: ${cpuInfo.model}`);
----

[[computergetdisplays]]
== computer.getDisplays()

返回所有连接显示器的信息。

[[return-object-awaited-4]]
=== 返回对象（异步）：

`Display` 对象的数组。

[[display]]
=== Display

* `id` 数字：虚拟显示器标识符。
* `resolution` 对象：显示器分辨率信息。
** `width` 数字：显示器宽度。
** `height` 数字：显示器高度。
* `dpi` 数字：DPI（每英寸点数）值。
* `bpp` 数字：BPP（每像素位数）值（也称为色深）。
* `refreshRate` 数字：刷新率，单位为赫兹（Hz）。

[source,javascript]
----
let displays = await Neutralino.computer.getDisplays();
for(let display of displays) {
    console.log(display);
}
----

[[computergetmouseposition]]
== computer.getMousePosition()

返回当前鼠标光标位置。

[[return-object-awaited-5]]
=== 返回对象（异步）：

* `x` 数字：距离屏幕左侧的像素距离。
* `y` 数字：距离屏幕上侧的像素距离。

[source,javascript]
----
let pos = await Neutralino.computer.getMousePosition();
console.log(`Pos: ${pos.x}, ${pos.y}`);
----
