== 成为 PMC 成员

PMC（项目管理委员会）成员通常会就项目未来做出决策。现有的 PMC 成员会根据活跃提交者的贡献情况，定期将他们加入 PMC 名单。 如果需要更详细的翻译，请告诉我！
